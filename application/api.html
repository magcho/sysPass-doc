API¶

sysPassのAPIは JSON-RPC v2 に基づいて設計されています。

APIのエンドポイントは “https://server_name/api.php” です

Example of JSON-RPC payload:

{
  "jsonrpc": "2.0",
  "method": "account/search",
  "params": {
    "authToken": "auth_token_for_api"
  },
  "id": 1
}
