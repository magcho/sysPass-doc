Authentication(認証)¶

sysPassでは何通りかの認証方式を利用することができます

  • MySQL/MariaDB database (デフォルト)

  • LDAP directory (OpenLDAP, eDirectory, Active Directory, freeIPA, etc)

注釈

LDAPオプションを有効化している時、何らかの理由でLDAPサービスが利用できない場合は大体の認証方式としてデフォルトのMySQL/MariaDB detabase方式に切り替わります。

データベースを用いる認証方式ではパスワードを元に Blowfish アルゴリズムでハッシュ化して検証しているため 決してパスワードは保存していません。

LDAP方式の場合

  • 前途の通りLDAPが利用できないときにデータベース方式に切り替えるため、データベース方式と同様に Blowfish アルゴリズムでハッシュ化したものをデータベースに保存しています。

  • ユーザーのログイン名・名前・メールアドレスを後から変更することはできません。
