LDAPの設定¶

Active Directory¶

Tips¶

  • アクセスしてきたユーザーがAccount Operatorsグループに属しているユーザーかどうかを確認します

OpenLDAP¶

OpenLDAPをセットアップするにはhttps://wiki.debian.org/LDAP/OpenLDAPSetupなどを参考にしてください。ここではDebian系のシステムにセットアップする手順を説明します。

OpenLDAPでグループメンバーシップ機能を使用するには「memberof」というoverlayを追加する必要があります。これはグループに属するユーザーに内部属性を追加するモジュールです。

以下でモジュールを設定する方法を説明します。

  • 以下の内容で ldap_memberof_add.ldif ファイルを作成してください

    dn: cn=module,cn=config
objectClass: olcModuleList
cn: module
olcModulePath: /usr/lib/ldap
olcModuleLoad: memberof

  • 以下の内容で ldap_memberof_config.ldif ファイルを作成してください

    dn: olcOverlay=memberof,olcDatabase={1}hdb,cn=config
objectClass: olcMemberOf
objectClass: olcOverlayConfig
objectClass: olcConfig
objectClass: top
olcOverlay: memberof
olcMemberOfDangling: ignore
olcMemberOfRefInt: TRUE
olcMemberOfGroupOC: groupOfNames
olcMemberOfMemberAD: member
olcMemberOfMemberOfAD: memberOf

  • 以下のコマンドを実行してLDAP構成を反映させます

    ldapadd -D cn=admin,cn=config -w "password" -H ldapi:/// -f memberof_add.ldif
ldapadd -D cn=admin,cn=config -w "password" -H ldapi:/// -f memberof_config.ldif

Tips¶

  • sysPassの’admin’ユーザーであることをOpenLDAPで確認できるようにするためには、このユーザーをsysPassへのアクセス権限を持つLDAPグループに追加する必要があります。

  • LDAPユーザーのユーザー名とメールアドレスは’displayname’・’fullname’・’mail’属性から生成されます。

  • Idapsに接続するには ’ldaps://mi_servidor_ldap’の形式で接続できます。

  • phpLDAPadmin をインストールすれば簡単にLDAPを管理することができます。
